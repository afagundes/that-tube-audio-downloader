# That Tube Downloader

Stream and download audio from YouTube videos.

## Usage

```shell
docker-compose up
```

## Usage in Development

Server:

```shell
cd server
node .
```

Client:

```shell
cd client
npm start
```

## Technologies

The server uses the following packages:

- Node
- Express
- Html-Entities
- Node-Fetch
- Youtube-Audio-Stream

The client uses the following packages:

- React
- React-Redux
- React-Icons
- Tailwind CSS
